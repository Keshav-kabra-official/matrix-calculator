A matrix-calculator project implemeted using C language.

Feature that can be done on user-given matrices:
>> Sum of matrics
>> Subtraction of matrices
>> Multiplication of matrices
>> Transpose of a matrix ( Tell if matrix is symmetric or not)
>> Determinent of matrix
>> Largest and Smallest number from a matrix
>> Minor and Co-factor of a matrix
>> Adjoint and Inverse of a matrix
